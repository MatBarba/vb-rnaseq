RNAseq DB Schema Documentation

This document describes the tables that make up the RNAseqDB schema. Tables are grouped into categories, and the purpose of each table is explained.

List of the tables:

SRA tables

  • study
  • experiment
  • sample
  • run

Taxonomy tables

  • species
  • strain

RNAseq Pipeline tables

  • file
  • analysis
  • analysis_param
  • analysis_file

Tracks tables

  • track
  • sra_track

Misc tables

  • publication
  • study_publication
  • drupal_node
  • drupal_node_track

Views

  • sra_to_track
  • sra_to_active_track

SRA tables

study
Show columns [Back to top]

Contains data defining SRA studies.

Column Type Default value Description Index
study_id INT(10) - SRA study id (primary key, internal identifier). key: study_id_idx
study_sra_acc CHAR(12) NULL SRA study accession (e.g. SRP000000). key: study_sra_acc_idx
study_private_acc CHAR(12) NULL Private study accession (e.g. VBSRP000000), for data without SRA accessions.
title TEXT NULL Title of the SRA study.
abstract TEXT NULL Abstract of the SRA study.
metasum CHAR(32) NULL Checksum of @study_sra_acc + @study_private_acc + @title + @abstract.
date TIMESTAMP CURRENT_TIMESTAMP Entry timestamp.
status ENUM:
  • ACTIVE
  • RETIRED
 'ACTIVE' Active (True) or retired (False) row.
experiment
Show columns [Back to top]

Contains data defining SRA experiments.

Column Type Default value Description Index
experiment_id INT(10) - SRA experiment id (primary key, internal identifier). key: experiment_id_idx
study_id INT(10) - Study primary id (foreign key). key: experiment_study_id_idx
experiment_sra_acc CHAR(12) NULL SRA experiment accession (e.g. SRX000000). key: experiment_sra_acc_idx
experiment_private_acc CHAR(12) NULL Private experiment accession (e.g. VBSRX000000), for data without SRA accessions.
title TEXT NULL Title of the SRA experiment.
metasum CHAR(32) NULL Checksum of @experiment_sra_acc + @experiment_private_acc + @title.
date TIMESTAMP CURRENT_TIMESTAMP Entry timestamp.
status ENUM:
  • ACTIVE
  • RETIRED
 'ACTIVE' Active (True) or retired (False) row.
sample
Show columns [Back to top]

Contains data defining SRA samples.

Column Type Default value Description Index
sample_id INT(10) - SRA sample id (primary key, internal identifier). key: sample_id_idx
sample_sra_acc CHAR(12) NULL SRA sample accession (e.g. SRS000000). key: sample_sra_acc_idx
sample_private_acc CHAR(12) NULL Private sample accession (e.g. VBSRS000000), for data without SRA accessions.
title TEXT NULL Title of the SRA sample.
description TEXT NULL Description of the SRA sample.
taxon_id INT(10) NULL NCBI taxon id.
strain TEXT NULL Name of the strain.
biosample_acc VARCHAR(15) NULL Biosample accession. key: biosample_acc_idx
biosample_group_acc VARCHAR(15) NULL Biosample group. key: biosample_group_acc_idx
strain_id INT(10) - Strain primary id (foreign key), to match the correct production_name.
label TEXT NULL Label for the sample, useful to find replicates.
metasum CHAR(32) NULL Checksum of @sample_sra_acc + @sample_private_acc + @title + @description + @taxon_id + @strain + @biosample_acc + @biosample_group_acc + @label.
date TIMESTAMP CURRENT_TIMESTAMP Entry timestamp.
status ENUM:
  • ACTIVE
  • RETIRED
 'ACTIVE' Active (True) or retired (False) row.
run
Show columns [Back to top]

Contains data defining SRA runs.

Column Type Default value Description Index
run_id INT(10) - SRA run id (primary key, internal identifier). key: run_id_idx
experiment_id INT(10) - Experiment primary key (Foreign key). key: run_experiment_id_idx
sample_id INT(10) - Sample primary key (Foreign key). key: run_sample_id_idx
run_sra_acc CHAR(12) NULL SRA run accession (e.g. SRR000000). key: run_sra_acc_idx
run_private_acc CHAR(12) NULL Private run accession (e.g. VBSRR000000), for data without SRA accessions.
title TEXT NULL Title of the SRA run.
submitter TEXT NULL Submitter name of the SRA run.
metasum CHAR(32) NULL Checksum of @run_sra_acc + @run_private_acc + @title + @submitter.
date TIMESTAMP CURRENT_TIMESTAMP Entry timestamp.
status ENUM:
  • ACTIVE
  • RETIRED
 'ACTIVE' Active (True) or retired (False) row.

metasum CHAR(32) NULL Checksum of @production_name + @strain.

status ENUM:
  • ACTIVE
  • RETIRED

RNAseq Pipeline tables

file
Show columns [Back to top]

Where all metadata about the files are stored.

Column Type Default value Description Index
file_id INT(10) - File id (primary key, internal identifier). key: file_id_idx
path TEXT - Path of the file.
type ENUM:
  • fastq
  • bam
  • bai
  • bed
  • bigwig
 NULL File type (fastq, bam...).
md5 CHAR(32) NULL md5 checksum of the file.
metasum CHAR(32) NULL Checksum of @path + @type + @md5.
date TIMESTAMP CURRENT_TIMESTAMP Entry timestamp.
status ENUM:
  • ACTIVE
  • RETIRED
 'ACTIVE' Active (True) or retired (False) row.

Show columns [Back to top]

Where all the analysis to create tracks are described.

Column Type Default value Description Index
analysis_id INT(10) - Analysis id (primary key, internal identifier). key: analysis_id_idx
name VARCHAR(32) NULL Name of the analysis.
description TEXT NULL Description of the analysis.
metasum CHAR(32) NULL Checksum of @name + @description.
date TIMESTAMP CURRENT_TIMESTAMP Entry timestamp.
status ENUM:
  • ACTIVE
  • RETIRED
 'ACTIVE' Active (True) or retired (False) row.
analysis_param
Show columns [Back to top]

Where the analysis parameters used to process every files are stored.

Column Type Default value Description Index
analysis_param_id INT(10) - Analysis parameters id (primary key, internal identifier). key: analysis_param_id_idx
analysis_id INT(10) NULL Analysis primary id (foreigh key). key: analysis_param_analysis_id_idx
program TEXT NULL Name of the Program used.
parameters TEXT NULL Complete command line parameters used.
metasum CHAR(32) NULL Checksum of @program + @parameters.
date TIMESTAMP CURRENT_TIMESTAMP Entry timestamp.
status ENUM:
  • ACTIVE
  • RETIRED
 'ACTIVE' Active (True) or retired (False) row.
analysis_file
Show columns [Back to top]

Linker between the tables file and analysis_parameter. Also links to the relevant runs.

Column Type Default value Description Index
analysis_file_id INT(10) - Analysis-file linker id (primary key, internal identifier). key: analysis_file_id_idx
analysis_param_id INT(10) NULL Analysis_parameter primary id (foreign key). key: analysis_file_analysis_param_id_idx
file_id INT(10) NULL File primary id (foreign key). key: analysis_file_file_id_idx
file_io ENUM:
  • INPUT
  • OUTPUT
 NULL If the file is an input or an output.
run_id INT(10) NULL Run primary id (foreign key).
metasum CHAR(32) NULL Checksum of @analysis_param_id + @file_id + @file_io + @scope + @scope_id

Tracks tables

track
Show columns [Back to top]

Where the tracks are stored, with a link to the corresponding file.

Column Type Default value Description Index
track_id INT(10) - Track id (primary key, internal identifier). key: track_id_idx
file_id INT(10) NULL File primary id (foreigh key). key: track_file_id_idx
title TEXT NULL Title of the track in E! genome browser.
description TEXT NULL Description of the track in E! genome browser.
metasum CHAR(32) NULL Checksum of @title + @description.
date TIMESTAMP CURRENT_TIMESTAMP Entry timestamp.
status ENUM:
  • ACTIVE
  • RETIRED
  • MERGED
 'ACTIVE' Active (True) or retired (False) row.
sra_track
Show columns [Back to top]

Defines what constitutes a track, i.e. one or several runs.

Column Type Default value Description Index
sra_track_id INT(10) - Track id (primary key, internal identifier). key: sra_track_id_idx
run_id INT(10) NULL SRA run primary id (foreign key). key: sra_track_run_id_idx
track_id INT(10) NULL Track table primary id (foreign key). key: sra_track_track_id_idx
date TIMESTAMP CURRENT_TIMESTAMP Entry timestamp.

Misc tables

publication
Show columns [Back to top]

Where the publication infos are stored.

Column Type Default value Description Index
publication_id INT(10) - Publication id (primary key, internal identifier) PRIMARY KEY. key: publication_id_idx

date TIMESTAMP CURRENT_TIMESTAMP Entry timestamp.
status ENUM:
  • ACTIVE
  • RETIRED
 'ACTIVE' Active (True) or retired (False) row.
